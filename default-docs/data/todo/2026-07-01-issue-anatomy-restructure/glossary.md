# Glossary

## Colour legend

Sidebar label tints (via `color:` frontmatter) mark discussion progress in
`brainstorm/01_overall-structure/`:

| Colour | Meaning |
|---|---|
| **Green** (`--color-success`) | Section design **settled** — decided, no open questions. |
| Untinted | Still open — carries a "To discuss" list. |

## Key terms

- **Graduation** — a resolved brainstorm's conclusion moving into `notes/` (the
  brainstorm stays, marked `**Resolved →** <target>`). Process → product.
- **Activity folder** — an agent-log `NNN_<code>_<name>/` folder holding one loop /
  audit / refactor / iteration / workflow run.
- **Kind** — what sort of work an entry is: a 2-letter code in agent-log folder names,
  a full word in brainstorm filenames.
