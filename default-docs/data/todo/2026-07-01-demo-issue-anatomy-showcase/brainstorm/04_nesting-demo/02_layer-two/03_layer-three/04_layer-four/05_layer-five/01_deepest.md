# Level 5 — the deepest allowed

Five folders below `brainstorm/`
(`04_nesting-demo/02_layer-two/03_layer-three/04_layer-four/05_layer-five/`). This is the
loader's maximum supported depth (`MAX_SUBFOLDER_DEPTH = 5`). Files may live here and
render — a **sixth** folder nested inside this one would be warned by the loader and
ignored. This is the ceiling; keep real trees to ~3.
