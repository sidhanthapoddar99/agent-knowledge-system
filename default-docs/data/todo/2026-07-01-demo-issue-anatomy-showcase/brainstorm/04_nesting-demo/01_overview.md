# Nesting demo — down to the 5-level hard cap

This multi-file brainstorm exists purely to exercise the loader's folder nesting. Content
subfolders may nest **up to 5 levels deep** (the hard cap); **up to 3 levels is the
recommended convention**. This demo deliberately walks to level 5 to prove the ceiling
renders as a nested `<details>` chain.

Walk down: `02_layer-two/ → 03_layer-three/ → 04_layer-four/ → 05_layer-five/` to the
deepest note. In real work you'd stop around level 3 and split into a sibling group instead.
